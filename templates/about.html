About the Application

This site uses a relational database to collaboratively build a compendium of all the people, places, and things that animate a role-playing campaign setting.

Players and Game Masters can join their own shared campaign in which everyone views the index of notable entries as their home page, and anyone can add new information.


SQL

Databases were established using sqlite3 from the Linux terminal. Error checking and adjusments were made utilizing phpLiteAdmin.

This database currently has tables for users, campaigns, parties, characters, places, items, quests, notes, and feedback.


Flask

The Flask microframework utilizes Python 3 and Jinja to dynamicaly generate HTML.


JavaScript

Interactivity, such as buttons that remain disabled until valid reponses are given, is made possible through JavaScript
